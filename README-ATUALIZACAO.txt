COMO SUBSTITUIR NO REPOSITÓRIO
1. Abra a pasta do seu projeto local.
2. Apague ou substitua os arquivos antigos:
   - index.html
   - style.css
3. Copie os novos arquivos desta pasta para dentro do projeto.
4. Crie a pasta assets, caso ela ainda não exista.
5. Copie as duas imagens para dentro da pasta assets.
6. Faça um teste abrindo o arquivo index.html no navegador.
7. Se estiver tudo certo, suba para o GitHub.
8. Depois publique novamente no serviço onde o site estiver hospedado.
